### Micro app Webhook
## Installation

```bash
$ npm install
```


## Setup your environment variables.

Make sure to update the new variables in the .env file
```bash
$ cp .sample.env .env
```

## Confirm your mongodb instance is on

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
